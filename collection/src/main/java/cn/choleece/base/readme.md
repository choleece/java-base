## Java 基本类型占内存大小
1. 整型

| 类型 |  内存大小    | bit数 |
|------|:----------:|------:|
| byte |  1B        | 1 x 8bit |
| short|  2B        | 2 x 8bit |
| int  |  4B        | 4 x 8bit |
| long |  8B        | 8 x 8bit |

2. 浮点

| 类型 | 内存大小     | bit数
|------|:------------|:--------|
| float| 4B          | 4 x 8bit|
|double | 8B         | 8 x 8bit|

3. char类型

| 类型 | 内存大小     | bit数
|------|:------------|:--------|
| char | 2B          | 2 x 8bit|

4. boolean类型

| 类型 | 内存大小     | bit数
|------|:------------|:--------|
| boolean | 1B       | 1 x 8bit|
